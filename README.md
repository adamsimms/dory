# Dory

Interactive 3D dory boat — live at [art.adamsimms.xyz/dory/](https://art.adamsimms.xyz/dory/), model on [Sketchfab](https://sketchfab.com/models/579758d4c7d3419cac6b380877ead206) (CC BY 4.0).

Related: [art.adamsimms.xyz](https://github.com/adamsimms/art.adamsimms.xyz), [Cloudberry](https://github.com/adamsimms/pinchards.is), [Adrift](https://github.com/adamsimms/adrift), [Waves](https://github.com/adamsimms/waves).

## Local

```bash
python3 -m http.server 8080
```

Open [http://localhost:8080/](http://localhost:8080/). No build step or secrets.
